#### 13. Bug修复
- ✅ **修复模型下载目录不一致问题**
  - 问题：模型下载时解压到context.getFilesDir()，但检查时使用getModelsStorageDir()
  - 修复：统一使用getModelsStorageDir()作为模型存储目录
  - 影响：修复后下载的模型可以正确识别和使用
  
- ✅ **修复在线模型检查问题**
  - 问题：在线模型（GLM-ASR、SenseVoice）不需要下载，但被当作离线模型检查
  - 修复：在isModelDownloaded()和相关检查中添加在线模型判断
  - 影响：在线模型可以直接使用，不会提示下载
  
- ✅ **优化按钮图标大小**
  - 问题：按钮图标太大导致文字换行
  - 修复：在SecondaryButton样式中添加iconSize="20dp"和padding
  - 影响：按钮文字不再换行，显示更美观

## 技术要点

### 1. 音频处理流程
```
视频文件 -> MediaExtractor提取音频 -> MediaCodec解码 -> 声道转换 -> 重采样 -> WAV格式(16kHz单声道PCM) -> 分段处理 -> Vosk识别 -> 文本
```

### 2. 语音识别模型
- **支持五种语音模型：**
  - Vosk中文小模型: vosk-model-small-cn-0.22 (50MB) - 速度快
  - Vosk中文标准模型: vosk-model-cn-0.22 (1.3GB) - 准确率高
  - Vosk中文多方言模型: vosk-model-cn-0.22-zh-cn (1.8GB) - 支持方言
  - GLM-ASR-Nano-2512 (在线) - 智谱AI出品，准确率高
  - SenseVoice (在线) - 阿里达摩院出品，准确率高
- **支持在线语音识别API**
  - 可配置自定义语音识别API
  - 支持科大讯飞、百度、阿里云等API
  - 未配置时自动使用离线模型
- 用户可在设置中选择和切换
- 首次使用时提示下载推荐模型
- 提供模型对比说明，帮助用户选择
- **模型存储位置：** 外部存储专用目录，更新应用不会删除

### 3. 大模型API
- 支持OpenAI兼容API
- 自动处理URL格式（添加/v1后缀）
- 支持Bearer Token认证

### 4. 权限处理
- Android 13以下需要存储权限
- 使用SAF选择文件，无需额外权限

### 5. 分享功能
- 使用Intent.ACTION_SEND
- 调用系统分享选择器
- 支持分享到任意应用
